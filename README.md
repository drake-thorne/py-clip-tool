# Movie Recap Tool (Python GUI)

A lightweight Python GUI tool for cutting a movie file into multiple short clips using FFmpeg. 
Useful for movie recap channels, commentary videos, or automated content workflows.

## Features
- GUI-based movie file selector
- Add/remove timestamp ranges
- Auto-export all clips using FFmpeg
- No manual video editing required

## Requirements
- Python 3.8+
- FFmpeg installed on your system (not a pip package)
  - macOS: brew install ffmpeg
  - Linux: sudo apt install ffmpeg
  - Windows: download from ffmpeg.org and add to PATH
